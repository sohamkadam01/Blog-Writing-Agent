Install Aletheia

Follow these 6 simple steps to start verifying websites instantly

1

📥 Download & Extract the Extension

The ZIP file has been downloaded to your computer. Right-click on aletheia_extension.zip and select "Extract All" to unzip the folder. Make sure to remember where you extract it!

2

🌐 Open Chrome Extensions Page

In your Chrome browser, type chrome://extensions in the address bar and press Enter. This will take you to the extensions management page.

3

⚙️ Enable Developer Mode

Look for the "Developer mode" toggle switch in the top-right corner of the extensions page and turn it ON. This allows you to load unpacked extensions.

4

📂 Load the Unpacked Extension

Click the "Load unpacked" button that appears after enabling Developer Mode. Navigate to and select the extracted aletheia_extension folder you created in Step 1.

5

📌 Pin Aletheia to Your Toolbar

Click the puzzle icon in Chrome's toolbar, find Aletheia in the list, and click the pin icon next to it. This keeps Aletheia always accessible.

6

🚀 Start Using Aletheia!

Visit any website, click the Aletheia icon in your toolbar, and get instant AI-powered insights, trust scores, and bias analysis on the content you're reading.
